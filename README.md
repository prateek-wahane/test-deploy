# Intelliware Global — Corporate Website

Enterprise-grade corporate website for **Intelliware Global Consulting and Services LLP**, built with Next.js 14+ App Router, Tailwind CSS, Framer Motion, and Lucide-React.

## Tech Stack

- **Framework:** Next.js 14+ (App Router)
- **Styling:** Tailwind CSS with custom design tokens
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Typography:** Plus Jakarta Sans (headings) + Inter (body)
- **Language:** TypeScript (strict mode)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Site Structure

| Route | Page |
|---|---|
| `/` | Home — Hero carousel, services, industries, global presence |
| `/services/[slug]` | 11 service detail pages |
| `/industries/[slug]` | 10 industry detail pages |
| `/about/[slug]` | About Us, Aspiration, Brand, Values, Leadership |
| `/careers` | Region-filtered job board (UK, Middle East, India) |
| `/contact` | Contact form with interactive office map |

## Design System

- **Primary:** Deep Navy `#0A192F`
- **Accent:** Cobalt Blue `#2563EB`
- **Surface:** Glassmorphism (blur + transparency)
- **RTL Ready:** Arabic language support with directional context
